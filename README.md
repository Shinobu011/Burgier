# Burgier
A modern web application that gives you valuable information about different kind of fast food, featuring an AI-powered assistant to help you with food stuff.

---

## **Features:**
- AI Assistant
- Modern UI
- Recipes Categories (Popular, Healthy, Budget-Friendly, 10-Minute Meals, and High-Protein Burgers)
- Ingredient-based search

## **Installation:**

### 1. Clone the repository:

```bash
git clone https://github.com/Shinobu011/burgier.git
cd burgier
```

### 2. Install dependencies
```bash
npm install
```

### 3. Create a .env file

```bash
API_KEY=YOUR_GOOGLE_GENAI_KEY
```

### 4. Start the server
```bash
node server.js
```

### 5. Open the website
```bash
http://localhost:8795
```
